รายละเอียดโครงการ: ระบบจัดการพัสดุคอมพิวเตอร์ออนไลน์

🛠️ เครื่องมือที่ใช้งาน: HTML5, CSS3, Google Apps Script, Google Sheets Database, Line Notify Integration

การอธิบายผลงานตามกรอบ STAR:

Situation (สถานการณ์): การแจ้งพัสดุชำรุดและการบำรุงรักษาในองค์กรเดิมทำบนแผ่นกระดาษ ทำให้สูญหายและติดตามสถานะยากลำบากมาก

Task (เป้าหมาย): เพื่อวางระบบไอทีซัพพอร์ตอัตโนมัติที่ช่วยให้ฝ่ายเทคนิคทำงานสะดวกขึ้นและสถิติไม่ตกหล่น

Action (การดำเนินงาน): ออกแบบเว็บแอปพลิเคชันจัดการพัสดุคอมพิวเตอร์ออนไลน์โดยใช้ QR Code ติดตั้งประจำห้องปฏิบัติการ เพื่อเปิดให้ผู้ใช้สามารถแจ้งเหตุได้ผ่าน Line Notify ทันที และระบบจะอัปเดตข้อมูลขึ้นคลังกลางโดยอัตโนมัติ

Result (ผลลัพธ์): ลดเวลาประสานงานและการซ่อมบำรุงลงถึง 65% คณะกรรมการสามารถตรวจสอบสถานะได้ตลอด 24 ชั่วโมง และข้อมูลพัสดุถูกต้อง 100%

อบรมความปลอดภัยไซเบอร์ในชุมชน

🛠️ เครื่องมือที่ใช้งาน: Microsoft PowerPoint, Canva Presentation, Simulated Phishing Platform

การอธิบายผลงานตามกรอบ STAR:

Situation (สถานการณ์): ชุมชนและเจ้าหน้าที่สถาบันยังขาดทักษะการแยกแยะลิงก์หลอกลวง (Phishing) และความรู้ตาม พ.ร.บ. คุ้มครองข้อมูลส่วนบุคคล (PDPA)

Task (เป้าหมาย): อบรมให้เกิดความตระหนักรู้และลดโอกาสการเกิดอาชญากรรมทางออนไลน์อย่างเป็นรูปธรรม

Action (การดำเนินงาน): วางแผนร่วมกับสมาคมไอทีจัดทำเนื้อหาบรรยายแบบเข้าใจง่าย และจัดจำลองแคมเปญเมลฟิชชิ่งทดสอบ (Interactive Simulation) เพื่อให้ผู้เข้าร่วมตระหนักถึงวิธีการกรอกข้อมูลอย่างปลอดภัย

Result (ผลลัพธ์): มีผู้เข้าร่วมอบรมกว่า 150 ราย อัตราการกดลิงก์หลอกลวงตัวอย่างลดลงเหลือไม่ถึง 5% ในการทดสอบหลังการอบรม และผลประเมินความพึงพอใจการบรรยายเฉลี่ยอยู่ที่ 4.85 จาก 5.00 คะแนน

การอธิบายผลงานตามกรอบ STAR:

Situation (สถานการณ์): นักเรียนขาดที่ปรึกษาทางเทคนิคเพื่อปูพื้นฐานการเขียนโมเดลวิเคราะห์ข้อมูลคาร์บอนเครดิตที่มีประโยชน์ในการประกวดนวัตกรรมระดับประเทศ

Task (เป้าหมาย): พัฒนาเยาวชนให้มีความรู้พื้นฐานและผลักดันให้ผลงานโครงงานคว้ารางวัลระดับภูมิภาค

Action (การดำเนินงาน): สอนการใช้กระบวนการ Design Thinking เพื่อหา Pain Point และฝึกอบรมการใช้งานคลังความรู้โมเดล Machine Learning อย่างง่าย เพื่อให้นักเรียนสามารถวิเคราะห์ภาพขยะและคำนวณคาร์บอนผ่านอินเทอร์เฟสที่เป็นมิตร

Result (ผลลัพธ์): ทีมนักเรียนสามารถคว้า **รางวัลชนะเลิศอันดับ 1 ระดับภาคใต้** และต่อยอดโมเดลไปใช้งานในกิจกรรมโรงเรียนคาร์บอนต่ำได้อย่างเห็นผล
