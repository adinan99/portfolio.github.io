ประวัติและใบรับรองความเชี่ยวชาญ

ประวัติการศึกษา ประสบการณ์ทำงาน และการเตรียมความพร้อมทางวิชาการคอมพิวเตอร์

ประวัติการทำงานและการศึกษา

มุ่งมั่นพัฒนาการจัดการระบบไอทีเชิงรับและเชิงรุกในองค์กรสุขภาพ โดยผสานทักษะจากปริญญาเทคโนโลยีสารสนเทศ และประสบการณ์วิชาชีพในฐานะนักวิชาการคอมพิวเตอร์ คณะวิทยาศาสตร์เทคโนโลยีและการเกษตร เพื่อสอดรับเป้าหมายการเปลี่ยนผ่านสู่ดิจิทัลสุขภาพ (Digital Health Transition)

การศึกษา วิทยาศาสตรบัณฑิต (เทคโนโลยีสารสนเทศ) มหาวิทยาลัยราชภัฏยะลา
ตำแหน่งงานปัจจุบัน นักวิชาการคอมพิวเตอร์ คณะวิทยาศาสตร์เทคโนโลยีและการเกษตร มหาวิทยาลัยราชภัฏยะลา

กรอบแนวคิดแบบ STAR Framework:

Situation (สถานการณ์): ระบบจัดการสารบรรณและการออกเลขหนังสือราชการรับเข้า-ส่งออกของคณะวิทยาศาสตร์เทคโนโลยีและการเกษตรเดิมพบปัญหาความล่าช้า และเกิดการลงข้อมูลเลขซ้ำบ่อยครั้งเมื่อมีเจ้าหน้าที่ทำงานพร้อมกันแบบออฟไลน์

Task (เป้าหมาย): ออกแบบและพัฒนาระบบ Cloud Database สำหรับการออกเลขและลงทะเบียนหนังสือราชการโดยเชื่อมต่อสคริปต์อัตโนมัติ (Apps Script) เพื่อลดอัตราลงทะเบียนซ้ำซ้อนให้เหลือ 0% และประมวลผลได้อย่างรวดเร็วทันที

Action (การดำเนินงาน): ผมในบทบาทผู้พัฒนาร่วม ได้ทำการออกแบบพัฒนาโครงสร้างเว็บหน้าจอ (Web Interface) และจำลองโครงสร้างฐานข้อมูลผ่าน Google Sheets API รวมถึงระบบตรวจสอบป้องกันความซ้ำซ้อนของการออกเลขหนังสือพร้อมกันในแบบเรียลไทม์

Result (ผลลัพธ์): ได้รับรางวัลรองชนะเลิศในการประกวดนวัตกรรม **KM STA FORUM 2025** ระบบดังกล่าวลดเวลาบันทึกและสืบค้นเอกสารลงถึง 80% ปลอดภัยสูง ค้นหาได้ทันใจ และได้รับคำชมอย่างยิ่งจากคณะผู้บริหารคณะฯ

ค่ายคอมพิวเตอร์สัมพันธ์ ซ่อมบำรุงเครือข่าย รร.บ้านวังพระเคียน

ดำเนินการเมื่อ 15 กุมภาพันธ์ 2568

กรอบแนวคิดแบบ STAR Framework:

Situation (สถานการณ์): ห้องปฏิบัติการคอมพิวเตอร์เพื่อการเรียนรู้ของโรงเรียนบ้านวังพระเคียนเผชิญปัญหาเครื่องคอมพิวเตอร์ขัดข้อง ฮาร์ดแวร์ชำรุด และสัญญาณโครงข่าย LAN ใช้การไม่ได้รวมกันกว่า 60% ของอุปกรณ์ทั้งหมด

Task (เป้าหมาย): ฟื้นฟู คลีน ตรวจเช็คฮาร์ดแวร์ และเซ็ตอัพเครือข่ายห้องเรียนคอมพิวเตอร์ให้สามารถกลับมาใช้งานเพื่อสนับสนุนการศึกษาให้แก่นักเรียนอย่างปลอดภัยราบรื่น

Action (การดำเนินงาน): ผมในบทบาทนักวิชาการคอมพิวเตอร์ ได้วางแผนจำแนกหมวดหมู่อาการเสียร่วมกับทีมนักศึกษา เข้าตรวจสอบระบบไฟฟ้า ทำความสะอาดพัดลมระบายความร้อน บันทึกประวัติและติดตั้งไดรเวอร์ที่จำเป็น ตลอดจนแก้ปัญหาการเชื่อมต่อ Access Point

Result (ผลลัพธ์): ฟื้นสภาพเครื่องกลับมาพร้อมใช้งานได้สมบูรณ์กว่า 95% ของห้องแล็บ ช่วยให้ทางโรงเรียนประหยัดงบประมาณการจัดจ้าง และนักเรียนสามารถใช้เรียนรู้เทคโนโลยีดิจิทัลได้อย่างมีประสิทธิภาพทันการศึกษา

ครูที่ปรึกษา: รางวัลรองชนะเลิศการแข่งขันหุ่นยนต์

โครงการพัฒนาทักษะสะเต็มศึกษา ประจำปีงบประมาณ พ.ศ. 2569

🏆 ความสำเร็จของลูกศิษย์: ด.ช.พีรณัฐ แซ่ตั้ง และ ด.ช.กวินภพ แก้วมณี (ม.1 รร.สาธิต มรภ.ยะลา ฝ่ายมัธยม)

กรอบแนวคิดแบบ STAR Framework:

Situation (สถานการณ์): รร.สาธิต มรภ.ยะลา มุ่งสนับสนุนผลงานนวัตกรรมรักษ์โลกของเยาวชนเพื่อเข้าสู่เวทีประชันทักษะระดับพื้นที่ แต่ทีมผู้เข้าแข่งขันในระดับ ม.1 ยังขาดความเข้าใจเชิงเทคนิคการประยุกต์บอร์ดสมองกลและการเขียนสคริปต์เซ็นเซอร์แยกแยะขยะ

Task (เป้าหมาย): แนะนำ อบรม ให้คำปรึกษา ออกแบบแนวคิด และแก้ไขโค้ดโปรแกรมควบคุมเพื่อให้หุ่นยนต์สามารถคัดกรองขยะตามประเภทได้อย่างแม่นยำภายในข้อจำกัดด้านเวลาประเมินสัมภาษณ์

Action (การดำเนินงาน): ผมปฏิบัติภารกิจเป็นโค้ชหลักด้านสมองกลเชิงลึก สอนหลักการตรวจจับความหนาแน่นและวัสดุของวัตถุ ฝึกอบรมการเขียนโปรแกรมลูปควบคุมและร่วมซ้อมกติกาในสนามแข่งขันเสมือนจริง

Result (ผลลัพธ์): นักเรียนสามารถคว้ารางวัลชนะเลิศ **รองชนะเลิศอันดับ 2 ระดับพื้นที่** ในการแข่งขัน STEM Education Robot Competition ถือเป็นเครื่องพิสูจน์ถึงความพร้อมในการส่งเสริมและการบูรณาการเทคโนโลยีระดับปฏิบัติงานได้เด่นชัด

วิทยากร: การเขียนโปรแกรมอุปกรณ์คอมพิวเตอร์และเครือข่ายไร้สาย

ดำเนินโครงการสัมมนาเริ่มต้น 4 เมษายน 2569 ณ ห้องปฏิบัติการคอมพิวเตอร์ อาคาร 6 (06-202) มรภ.ยะลา

💻 ขอบเขตเนื้อหา: การสร้างซอฟต์แวร์ประยุกต์เบื้องต้นบนโมบายดีไวซ์ และการเชื่อมโยง API ของข้อมูลเครือข่ายไร้สาย

กรอบแนวคิดแบบ STAR Framework:

Situation (สถานการณ์): นักศึกษาสาขาเทคโนโลยีสารสนเทศต้องการความรู้เพิ่มเติมเชิงประยุกต์ด้านแนวคิดการเชื่อมเครือข่ายไร้สายเพื่อพัฒนาเป็นโครงสร้างแอปพลิเคชันที่มีประโยชน์ในโลกยุคหลังวิกฤตความปลอดภัยดิจิทัล

Task (เป้าหมาย): อบรม ปฏิบัติการ ให้คำปรึกษา และสรุปใจความสำคัญของโปรแกรมควบคุมพอร์ตการสื่อสาร in 30 ชั่วโมงเต็มเพื่อให้นักศึกษาสามารถสร้าง Web Services ขนาดเล็กได้ด้วยตนเอง

Action (การดำเนินงาน): ปฏิบัติหน้าที่ผู้จัดการหลักสูตรและวิทยากรผู้บรรยาย นำเสนอบทเรียนตั้งแต่พื้นฐานสถาปัตยกรรมแบบไคลเอนต์-เซิร์ฟเวอร์ ออกแบบแบบฟอร์มเชิงโต้ตอบ และการใช้เครื่องมือควบคุมการตอบรับสัญญาณ

Result (ผลลัพธ์): นักศึกษากว่า 45 รายผ่านการทดสอบและเข้าใจในการเขียนสคริปต์ควบคุมอุปกรณ์เคลื่อนที่ สามารถไปต่อยอดพัฒนาโปรเจกต์งานเทคโนโลยีที่มีคุณภาพสูงได้

วิทยากรเสริม: การเลือกใช้เครื่องมือปัญญาประดิษฐ์เพื่อการสอน

ดำเนินกิจกรรม 12-13 มีนาคม 2569 ณ สพป.นราธิวาส เขต 1

🛡️ จุดมุ่งหมายด้านปลอดภัยไซเบอร์: มุ่งถ่ายทอดการกรอกและใช้งาน Generative AI บนกฎเกณฑ์พระราชบัญญัติข้อมูลสารสนเทศและความปลอดภัยส่วนบุคคล (PDPA)

กรอบแนวคิดแบบ STAR Framework:

Situation (สถานการณ์): ครูผู้สอนในศูนย์ประถมศึกษาหลายส่วนมีความต้องการนำนวัตกรรม AI ไปอำนวยความสะดวก แต่มีความเสี่ยงสูงต่อการอัปโหลดข้อมูลคนไข้หรือประวัตินักเรียนในฐานข้อมูลของคลาวด์สาธารณะโดยไม่คัดกรอง

Task (เป้าหมาย): อบรมเชิงปฏิบัติการให้ความรู้และสร้างความเข้าใจเชิงจริยธรรมความปลอดภัยด้านคอมพิวเตอร์และการตรวจคัดกรองเนื้อหาหลุดรั่ว

Action (การดำเนินงาน): ปฏิบัติงานเป็นวิทยากรบรรยายร่วมกับทีมศูนย์ปัญญาประดิษฐ์ (YRU AI Center) แนะนำเครื่องมือปัญญาประดิษฐ์ที่ปลอดภัย รวมถึงสาธิตการใช้ Sandbox และการตรวจสอบความสมบูรณ์ของประวัติเชิงลึก

Result (ผลลัพธ์): ผู้เข้าร่วมสามารถบูรณาการสื่อสร้างความปลอดภัยข้อมูลในการเรียนรู้อย่างรอบด้าน และสร้างความตระหนักต่อภัยไซเบอร์ได้เป็นเลิศสอดรับตามมาตรฐานกระทรวงดิจิทัล

สัมมนา เรื่อง เส้นทางสู่ความพร้อมก่อนออกฝึกสหกิจ

ดำเนินการเมื่อ 17 กันยายน พ.ศ. 2568 ณ ประชุมติ่งเชียง อาคาร 24 มรภ.ยะลา

กรอบแนวคิดแบบ STAR Framework:

Situation (สถานการณ์): นักศึกษาไอทีก่อนออกฝึกสหกิจมักประสบปัญหาช่องว่างด้าน Soft Skills เช่น การแก้ปัญหาเฉพาะหน้า การสื่อสารข้ามแผนก และการทำงานร่วมกับผู้ใช้ระบบที่ไม่มีทักษะคอมพิวเตอร์

Task (เป้าหมาย): อภิปรายและบรรยายหัวข้อสำคัญ "Soft Skills for IT Students" นำเสนอทักษะความยืดหยุ่นและการทำงานร่วมกันเพื่อให้ผู้เข้าร่วมสัมมนามีความพร้อมทำงานในองค์กรจริง

Action (การดำเนินงาน): ผมและทีมวิทยากรได้จัดทำสไลด์และจำลองสถานการณ์จำลอง (Roleplay Case Study) เช่น การอธิบายเรื่องเน็ตเวิร์กช้าให้แผนกบริหารเข้าใจ เพื่อกระตุ้นความเข้าใจและร่วมตอบคำถาม

Result (ผลลัพธ์): นักศึกษามีความตระหนักต่อกระบวนการทำงานร่วมกับสหวิชาชีพเพิ่มขึ้นอย่างเห็นได้ชัด ได้รับผลประเมินพึงพอใจการบรรยายสัมมนาระดับยอดเยี่ยม
